## 作品集

### Dexter – 智能辅助机器人

**项目简介：**  
Dexter 是一款专为辅助教育工作者设计的智能机器人，能够高效完成白板清洁、图表模板绘制以及重要内容拍摄。

**主要特点：**

- **智能化：** 利用机器学习技术优化清洁路径，实现精准导航。
- **可靠性：** 稳健的轨道系统确保在白板上准确定位与移动。
- **创新性：** 兼具绘制与擦除功能，拓展了机器人的应用场景。
- **多功能性：** 除了核心清洁功能外，还支持模板绘制与内容拍摄，方便学生访问。
- **便捷性：** 硬件与软件设计均以易用性为目标，快速上手无压力。
